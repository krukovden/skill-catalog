# productivity

Workflow tools that aren't tied to writing code — thinking, interviewing, authoring.

**Promoted**: skills here ship in `catalog.json` and the Claude plugin.

- **[grilling](./grilling/SKILL.md)** — Relentless interview that walks the decision tree one question at a time, looking facts up itself and putting only the decisions to you, until there's shared understanding. Reach for it before building anything whose requirements are still fuzzy.
- **[writing-great-skills](./writing-great-skills/SKILL.md)** *(user-invoked — type it; the agent never fires it)* — Reference for authoring and pruning skills: the invocation trade-off (context load vs cognitive load), the information hierarchy and progressive disclosure, completion criteria, leading words, and the six failure modes that make a skill unpredictable. Full definitions in its [GLOSSARY.md](./writing-great-skills/GLOSSARY.md).

Both are adapted from [mattpocock/skills](https://github.com/mattpocock/skills) (MIT) — see [ATTRIBUTION.md](../../ATTRIBUTION.md).
