# in-progress

Skills still being built. Expect rough edges, breaking changes, and abandoned experiments.

**Not promoted**: nothing here reaches a user. `scanSkills()` defaults to the promoted
buckets, so these skills are absent from `catalog.json`, from `.claude-plugin/plugin.json`,
and from the root `README.md` — that is the whole point of the bucket. It lets an
unfinished skill be committed, reviewed, and evaluated without being shipped.

Graduate a skill by moving its folder into a promoted bucket (`engineering/`, `ops/`) and
running `npm run build`.
